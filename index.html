AlexDiBot - Groccery Prices Robot

What is this?

- This is a robot that gets daily data from leading groccery shops for a University project.

- Why?... Some people (like me) prefer to make a budget using a spreadsheet and the website can be hard to navigate quickly and copy price info each week. This makes budgeting a breeze. Also this data is useful for tracking prices overtime.

- Data is free and avaliable in CSV (can open this in excel or google sheets) file download.

- API endpoints are coming soon.
